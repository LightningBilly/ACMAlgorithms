题目链接 ：http://poj.org/problem?id=1243

题目大意：
有一种猜数字游戏，从1到N中选一个数字（N是正整数），一开始玩家有G次猜测次数，以及L条命，然后每次玩家猜一个数字，每猜一次G减小1，如果猜对了就赢了，如果猜小了继续，如果猜大了L也要同时减小1。当G等0或L小于0时还未猜中，就输了。
提问：给定G和L的大小，问题N最大是多少，同时玩家一定可以有策略赢得游戏。

题解：
可以用动态规划来解决此题。
设dp[i][j]代表L=i, G=j时，玩家可以保证胜利的最大的连续区间长度。则dp[L][G]为所求答案。
显然dp[0][j]=j;
当i>0时，我们可先尝试着猜一次，假我们猜的数字是S，则最坏有两种情况，猜小了和猜大了：
1）猜小了，则剩下还有i不变, j=j-1，则接下来一定要保证获胜的区间长度为S-1，即dp[i][j-1] >=S-1。
2）猜大了，则剩下还有i=i-1, j=j-1, 则获胜区间的长度为S+dp[i-1][j-1]
综上所述，dp[i][j] =max( S+dp[i-1][j-1]) ( dp[i][j-1]>=S-1)。
可知 S<= dp[i][j-1]+1, 则dp[i][j] = dp[i][j-1]+1+dp[i-1][i-1].
转移方程如下：
dp[0][j]=j;
dp[i][j] = dp[i][j-1]+1+dp[i-1][i-1]. (j>0)
