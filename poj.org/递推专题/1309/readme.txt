题目大意
给定一个长度为n只有数字0和1组成的数组，从左到右分别编号1到n。
每次只能将一个数字进行变换（0到1，1到0），这个变化要遵循以下规则
1.每次只能变化一个数字
2.如果这个数字的编号是1，那么可以直接变化。
3.如果编号 1,...,k-1 (1<=k<n) 的数字都为0且 编号k的数字为1，那么编号k+1的数字可以变化。

给定数组，问要多少次变化才能将数组中的数字都变成0.

题目分析：
先讨论如果在数组中只有一个1存在的情况。
用dp(i)表示编号为i的1要变成0，要用多少次变化。
显然i=1时，dp(1) = 1，
dp(2)=dp(1)*2+1;
相当于先把前边的0变成1，然后将自己变成0，再将前面的变成0，
所以 dp(1)=1, dp(i)=dp(i-1)*2+1, 由于i最多可以到达1000，
dp(1000) = 2^1000-1, 是一个很大的数，所以要用到大数运算。


当有多个1的时候，要把前面的1先变成0，但是在变化过程有存在抵消的情况。
举例如下，
1 0 1 0
要想把第2个1变成0，本来在第1个为0的情况下，把第3个数字变为1的变化次数为7，但是现在第1个数本来就是1，相当于我们可以利用已有的情况所以要减去dp(1) ，所以结果为7-1=6。

所以总结如下，
从后往遍历arr[i]，遇到arr[i]==1时，如果是遇到了第奇数个1，则加上dp(i), 否则减去dp(i)。

最后注意要用大数运算。
